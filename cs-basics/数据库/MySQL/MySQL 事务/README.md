### 事务日志

 [redo log.md](redo log.md) ：redo log 是 InnoDB 存储引擎层生成的物理日志，实现事务的持久性，主要用于掉电等故障恢复。

 [undo log.md](undo log.md) ：undo log 是 InnoDB 存储引擎层生成的逻辑日志，实现事务的原子性，主要用于事务回滚和 MVCC。
