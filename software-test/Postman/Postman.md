### 什么是 Poatman

Postman是一个用于构建和使用API的API平台。Postman简化了API生命周期的每个步骤并简化了协作，因此您可以更快地创建更好的API。

API 生命周期主要包括：

- design
- testing
- documentation
- mocking todiscovery.
